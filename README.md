# dont-litter-lava-level-and-ending-credits
Showcases the Lava Level and Ending Credits that I created as part of a 3D platformer and adventure game where the player explores various levels with different themes each and can run, jump, and attack. Game was developed as part of a 5-member team project in Georgia Tech's CS 6457: Video Game Design course using C# and the Unity Game Engine.
